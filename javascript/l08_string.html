문자의 집합인 문자열 타입 String

기본데이터 타입인 문자열은 배열의 일종으로 문자를 순서대로 저장한 data다. 이때 문자는 charset의 규칙(표)를 따른다.

  • 문자열 데이터 타입은 "",'',`` 을 사용해서 생성(선언)할 수 있다.(리터럴 방식)
  • 참조형의 구조(배열)를 따르는 문자열이 기본데이터 타입으로 사용되는 이유는 프로그래밍에서 가장 많이 사용되고 있끼 때문이고 배열이 참조하는 문자 data가 문자표의 형식을 따르기 때문에 형식을 갖주기 좋다.
  • 문자열은 배열이기 때문에 index를 key로 참조할 수 있다 예) "가나다라마바사아자카타"[3] :
  • 문자열 기본데이터 타입의 wrapper 객체는 String 이다.
  • 문자열을 인스턴스 객체로 생성하는 것을 권장하지 않는다.(기본데이터타입 모두)

문자열 작성 방식1

  • "안녕하세요 '최경민' 입니다." :
  • '안녕하세요 "최경민" 입니다.'
  • "안녕하세요 "최경민" 입니다." (x)
  • '안녕하세요 '최경민' 입니다.' (x)
  • "안녕하세요 \"최경민\" 입니다." :
  • '안녕하세요 \'최경민\' 입니다.' :
  • `안녕하세요 "최경민" '36' 입니다.` (explorer x):

문자열 작성 방식2(문자열 + 연산)

  • 문자열은 + 연산을 할 수 있다.(배열의 길이를 더한다.)
  • "a"+"b" =
  • "a"-"b" =
  • "a"*"b" =
  • "a"/"b" =
  • "10"+1 =>"10"+new Number(1).toString()=
  • "10"+true =
  • "10"+NaN =
  • "10"+undefined =
  • "10"+null =
  • "10"+window =>"10"+window.toString()
  • "10"*3 =>Number("10")*3=
  • "저의 나이는 "+ (2021-1986+1) +" 입니다."
    "저의 나이는 "+ (36) +" 입니다."
    "저의 나이는 "+ "36" +" 입니다." :
  • `저의 나이는 ${2021-1986+1} 입니다.` :
