자바스크립트의 논리 연산자

&&(and) ||(or) !(not)

|| (or) 합집합을 의미하는 or(더하기) 연산자

  • 사용법: ( (boolean) || (boolan) ) return boolan
  • boolean: "있다", null, >, <, == ,=== ,! !=, !== , A instanceof B, ....
  • 둘 중에 하나가 true이면 true를 반환한다.
  • 둘다 false일 때 false를 반환한다.
  • true가 있는지 검사하는 연산 flase||true||false||false||false
  • 1+0+0+0+0=1

&& (and) 교집합을 의미하는 and(곱하기) 연산자

  • 사용법: ( (boolean) && (boolan) ) return boolan
  • boolean: "있다", null, >, <, == ,=== ,! !=, !== , A instanceof B, ....
  • 둘 중에 하나가 false이면 false를 반환한다.
  • 둘다 true일 때 true를 반환한다.
  • false가 있는지 검사하는 연산 true&&true&&true&&true&&false
  • 1*1*1*1*0*1*1*1=0

연산의 우선 순위

&& 곱하기 연산으로 || or(더하기) 연산보다 우선순위를 갖는다.

() > 증감(++,--) > 산술(*\% > +-) > 비교 > 논리(&&>||) > 대입

  • true && false || true && false :
  • true && true || true && false :
  • true || true && true || false :
  • true || false && true || false :
  • false || false && true || false :
  • NaN || 2 :
  • NaN && 2 :
  • NaN || 2 && null :
  • NaN && 2 && null :
  • a가 10보다크고 5보다 작다 :
  • a가 5보다크고 10보다 작은 것과 0보다 크고 3보다 작은 것 :

! not 연산

  • !true=false, !false=true
  • !!true=true, !!false=false
  • !(a>0 && a<1) == (a<=0 || a>=1)
