참조형 데이터 타입과 배열과 Map과 Set

  • 참조형 데이터 타입 : 모든 객체, 여러 값을 참조하는 type, Object의 type으로 데이터 저장
  • Object : 모든 객체의 부모type, 객체를 생성하는 방법을 제공(new,constructor),객체에 속성을 지정(this)
  • Object 선언법 : new Object()[property]="" , {property: value}, function Obj{this.a=""} new Obj();
  • Array : 프로퍼티가 index인 자료구조로 반복문을 실행하고 길이를 알수 있는 참조형 데이터 타입.
  • Array 선언법 : new Array(1,2,3), [1,2,3]
  • Map : 필드(this.a=1)에 index를 추가한 자료구조로 프로퍼티를 모든 데이터 타입으로 사용가능한 참조형 데이터 타입이다.
  • Set : 필드의 data 중복을 제거하는 자료구조로 data의 순서는 존재하지만 프로퍼티(key)가 중요하지 않은 참조형 데이터 타입이다.
