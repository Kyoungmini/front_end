비교 연산자 return boolean

boolean true(참) false(거짓)을 표현하는 data로 비교 연산자의 결과 값이다.

operator 이름 설명 예시
a==b 동등비교 a와 b의 값이 같은지를 비교 10==10 :
10==new Number(10) :
10=="10" :
a===b 엄격한 동등비교 a와 b의 값이 같고 (참조형은 값이 없다.=>복수의 데이터를 구조화하는 pototype을 참조한다.) 타입도 같은지 비교 10==="10" :
10===new Number(10) :
new Number(10) === new Number(10) :
a != b(! not) 비동등 a와 b의 값이 같지 않은지 비교 10 =! 9 :
10 =! "10" :
10 =! new String("10") :
a !== b 엄격한 비동등
a > b 초과(크다)
a < b 미만(작다)
a >= b 이상
a <= b 이하

논리 연산자(&& and,|| or,! not)

합집합을 나타내는 || (or) 연산자

교집합을 나타내는 && (and) 연산자

차집합을 나타내는 ! (not) 연산자
