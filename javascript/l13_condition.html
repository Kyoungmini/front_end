조건문 if 작성법

  • if(조건문){실행; block,지역...} : 조건이 true일때 블럭이 실행된다.
  • if는 꼭 소문자로 작성한다. IF (x)
  • 실행이 한줄일때 {}를 생략할 수 있다. if(조건)실행;
  • 조건 연산자로 ==,=!,!,===,!==,>,<,>=,<=,isArray(),등을 사용할 수 있다.
  • (data)안에 조건 연산자를 사용하지 않으면 => Boolean(data)

else if(), else 조건이 참이 아닐 때도 실행해 보자

  • if(조건문){true 실행}else{ false 실행}
  • if(조건문){true 실행}else if(조건문){ false 실행}else{나머지}
  • default 값이 else 대신할 수 있다.
