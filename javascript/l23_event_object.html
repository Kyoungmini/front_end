이벤트 발생시 이벤트 정보를 담는 Event Object

MouseEvent Object 마우스 이벤트가 발생시 생성되는 객체

  • MouseEvent.button : 이벤트 발생 시 누른 버튼 번호(0:왼쪽,1:휠,2:오른쪽)=>누르지 않은 상태 확인x
  • MouseEvent.whitch : 이벤트 발생 시 누른 버튼 번호(0:누르지않음,1:왼쪽,2:휠,3:오른쪽)
  • MouseEvent.buttons : 이벤트 발생 시 누른 버튼 번호(0:누르지않음,1:왼쪽,2:오른쪽,3:왼쪽+오른쪽,4휠,5~7:조합,8:뒤로가기, 16:앞으로가기)

  • screenX,Y :

  • offsetX,Y : 이벤트가 발생한 노드를 기준으로 마우스 포인트 좌표
