기본데이터 타입 string의 래퍼 객체인 String의 메소드와 속성(property)를 알아보자

String의 속성

  • "문자열".length : 문자열의 길이(1부터시작)

String 메소드

"문자열".charAt() == "문자열"[] :

"문자열".charCodeAt() (문자의 유니코드 번호) :

"문자열".concat("문자열2","문자열3",...) == "문자열"+"문자열2"+"문자열3"...: 다하기 연산

" 문자열 ".trim() == "문자열" : 공백제거

"가,나,다,라".split(",") == ["가","나","다","라"] : 특정문자를 기준으로 문자열을 자르고 배열로 변환한다.
