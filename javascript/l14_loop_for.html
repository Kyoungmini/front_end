자바스크립트의 반복문

for 반복문

for(초기문; 조건문; 증감문){조건문이 true일 때 반복 실행}

  • 초기문: let i=0; i는 반복이 처음 실행될 때 한번만 선언된다.
  • 조건문: if(조건문)과 같이 Boolean 실행되는 곳으로 조건이 false이면 반복문은 끝이 난다.
  • 증감문: 초기값 i를 증가 또는 감소 시켜 조건이 false가 되어 벗어나도록 제한을 거는 곳 예) 100번 실행하는 반복문 for(let i=0; i>-100; i--)
  • for 반복문은 while 반복문에서 초기문,조건문,증감문을 조건문 작성하는 곳에 모두 모아 놓은 형태

while 반복문

let i=0;(초기문) while(조건문){조건이 true일 때 반복 실행; 증감문;}
