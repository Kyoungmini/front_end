자바스크립트의 데이터 타입을 알아보자

자바스크립트에서 사용되는 data는 모두 각자의 type 갖는다. 그 타입은 크게 2가지로 나뉘는데 기본형과 참조형으로 나눈다.

기본 데이터 타입

프로그래밍에서 가장 많이 사용되는 데이터를 타입으로 나타낸 것으로 대부분이 저장된 메모리의 조수를 바로 참조한다(값이 존재한다).

  • 기본 데이터 타입은 원시 값(primitive)이라 불리고 값이 저장된 메모리를 바로 참조한다.
  • 원시 값은 변경될 수 없는 값으로 한번 할당된 값은 변경되지 않는다.
  • 원시 값은 추측할 수 있는 값(많이 사용되고 수이다)으로 구성되어 있어서 메모리에 미리 할당되어 있는 것들 있다.
  • 메모리에 미리 저장되어 있고 data가 가벼워서 쓰고 읽는 것이 빠르다.

기본 데이터 타입의 종류

  • string :문자열(텍스트)를 저장하는 type으로 배열과 구조는 같지만 기본데이터 타입으로 사용된다.
  • number : 소수를 포함수 수를 저장하는 type
  • bigint : number type으로 저장할 수 없는 큰 수를 저장하는 type
  • boolean : true or false 를 나타내는 type
  • undefined : 정의되지 않음 let a;
  • null : 자료(data)가 없음, 자료를 요청했을 때 그 자료가 없으면 null을 반환한다. let b=null;
  • symbol : 고유한 값을 반환하는 타입이다.

참조형 테이터 타입(기본형을 제외한 모든 data)

복수의 데이터(값)를 구조화(Prototype)해서 참조하고 있다. [10,22,33 ...]=> 참조형 데이터 타입은 값을 바로 참조하지 않는다.

가장 많이 사용되는 참조형 data

  • Array: 복수의 데이터를 index를 키로 참조하는 참조형 데이터 타입
  • Json: 복수의 데이터에 임의의 키를 지정해서 참조하는 참조형 데이터 타입(객체)
  • Math: 숫자의 연산(수학) 작업을 수행하는 객체
