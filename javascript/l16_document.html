DOM 을 참조하는 window.document 객체로 문서의 모든 요소를 접근하자.

html DOM(document Object Model)

  • DOM 은 html 문서를 객체로 만드는 모델로 트리 구조를 갖는다.
  • 이때 각 객체를 노드 개체 or 객체 or 요소라 부르며 트리구조를 참고해서 각 노드를 검색한다.
  • 자바스크립트로 각 노드는 수정 삭제가 가능하고 새로운 노드를 추가할 수도 있다.
  • DOM은 W3에 의해 표준이 지정되 있어 모든 브라우저가 동일한 모델을 사용한다.(웹표준)

DOM의 노드(요소)를 찾는 함수

  • document.getElementById("id") return element ==> element.style.color="red"
  • document.getElementsByTagName("tag") return HTMLCollection ==> HTMLCollection[0].style.color="red"
  • document.getElementsByClassName("class") return HTMLCollection ==> HTMLCollection[0].style.color="red"
  • document.querySelector("css selector") return element
  • document.querySelectorAll("css selector") return NodeList ==>NodeList[0].style.color="red"

DOM의 form(제출 양식) 요소 를 찾는 방법

form 요소의 이름은 id 처럼 document에서 유일한 이름이다.

  • form의 이름으로 form 노드개체를 찾는법
    1. document.forms["formName"]
    2. document.forms.formName
    3. formName
      1. form의 입력(input) 요소의 이름으로 입력요소 노드개체를 찾는법(입력요소의 이름은 form안에서 유일한 이름이다.)
        1. document.forms["formName"]["inputName"] : form이나 입력요소의 이름을 실행단계에서 받아서 사용할때 유용
        2. document.forms.formName.inputName
        3. formName.inputName

DOM을 조작하는 함수(Node 개체를 추가 삭제) dom을 추가

  • document.createElement("tag") return element: 요소의 이름으로 노드개체를 생성
  • document.element.append(element) :노드개체 내부의 뒤에 노드를 추가하는 함수
  • document.element.prepend(element) :노드개체 내부의 앞에 노드를 추가하는 함수
  • document.element.after(element) :노드개체 형제의 뒤에 노드를 추가하는 함수
  • document.element.before(element) :노드개체 형제의 뒤에 노드를 추가하는 함수
  • document.removeChild(element) : 노드를 삭제
  • document.element.write(text) : 노드개체에 text 노드 개체를 추가하는 함수 (element.innerText="text")
  • document.element.innerHTML="html"; : 노드개체의 내부에 새로운 노드 개체로 수정(html text로 수정됨)

노드 개체의 속성을 변경

  • element.attribute="value"; (href,id,title...)
  • element.setAttribute( "attribute" , "value") 실행단계에서 속성의 이름을 지정해서 변경할 수 있다.
  • element.style.property="value"; (backgroud-color ==> backgroudColor)

노드 개체에 이벤트 핸들러를 추가

  • element.onevent = function(event){실행코드}
  • html 요소내부에서 재정의 : element onevent="실행코드"
  • element.addEventListener("event",function(){} );
