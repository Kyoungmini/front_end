array : 객체 [1,2,3..] , Array : 내장객체(예약어)

array.pop() : 배열의 마지막 아이템을 반환하고 제거한다.

Array.push(item) : 배열의 마지막에 새 아이템을 추가한다.

array.shift() : 배열의 처음 아이템을 반환하고 제거한 후 모든 아이템을 한칸 당긴다.

Array.unshift(item) : 배열의 처음에 아이템을 추가하고 모든 아이템을 한칸 민다.

array[i] : 특정 index의 값을 바꾼다.

array[length](item의 길이보다 길때) : 특정 index의 값을 추가한다.

array.slice(start)

배열의 start 부터 끝까지 잘라서 새 배열을 반환

array.slice(start,end)

배열의 start 부터 end(길이)까지 잘라서 새 배열을 반환

array.splice(insert_delete_start, delete_length, item1,item2 ... )

배열의 start 부터 length(길이)까지 자른 후 start 지정에 item 추가

array.splice(insert_start, 0, item1,item2 ... )

배열의 start에 item 추가

array.splice(delete_start, delete_length)

배열의 start부터 length까지 삭제하고 반환
