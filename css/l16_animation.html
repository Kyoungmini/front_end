css animation으로 복작한 애니메이션을 만들어 보자

box를 클릭
  • transition으로 복잡한 애니메이션을 만들 수 없다.
  • 복잡한 애니메이션을 만들기 위해서는 animation 속성을 지정하고 @keyframes으로 frame마다 애니메션을 분활 적용해야한다.
box를 클릭

animation-fill-mode

animation-fill-mode: none

animation-fill-mode: forwards

animation-fill-mode: backwards

animation-fill-mode: both

animation 속성들

  • animation-name: @keyframes의 이름
  • animation-duration: @keyframes 0~100%까지 걸리는 시간
  • animation-timing-function: 애니메이션 속도 함수
  • animation-delay: 애니메이션이 시작을 대기하는 시간
  • animation-iteration-count: 반복 재생하는 횟수(default:1) 1~,infinite
  • animation-play-state: 재생 상태( paused, running ) 애니메이션의 이름을 노드에 추가하면 자동으로 running이 된다.
  • animation-direction : 애니메이션 진행 방향 (0~100%)
    • normal : from 0 => to 100
    • reverse : to => from
    • alternate(iteration-count>1) : 홀수 normal, 짝수 reverse
    • alternate-reverse(iteration-count>1) : 홀수 reverse, 짝수 normal
  • animation-fill-mode: keyframes의 from과 to 일때 상태를 결정
    • none : from 일때 from의 스타일 적용하지 않고 대기(delay)하다가 애니메이션이 시작될 때 from의 스타일 적용하면서 to까지 진행되고 to가 완료되면 애니메이션 시작 전으로 되돌린다.
    • forwards : from 이 none과 동일하고 애니메이션이 완료되면 to 지점에서 멈춘다.
    • backwards :from에서 적용될 css를 모두 적용하고 대기하다가 애니메이션을 실행하고 완료되면 애니메이션 시작 전으로 되돌린다.
    • both : from이 backwards와 같고 to가 forwards가 같다.
