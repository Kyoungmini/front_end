css float 블럭 정렬하기

  • float는 글자에 블럭을 삽입하기 위해 만들어짐
  • float가 적용된 노드는 글자로 취급받고 블럭의 부피가 무시당한다.
  • float의 부피가 무시당하기 때문에 부모노드가 float 노드를 인지하지 못하는 것으로 보인다.
  • float는 노드는 인접한(부모,형제) 다른 노드의 인라인만 간섭한다.
