  • none : 화면에 표시되지 않는 상태로 삭제는 아니다.

inline 사용시 주의할 점

  • inline 요소에 padding을 지정할 수 있지만 부모 노드가 padding을 인지하지 못한다.
  • inline 요소에 margin을 지정할 수 없고 설령 되더라도 부모 노드가 margin을 인지하지 못한다.
  • inline content 영역만 있다.
  • inline block 속성(padding,margin,border)을 사용하고 싶다면 inline-block으로 지정해라!!
