수치가 변경되면 발생하는 애니메이션 Transition

  • transition은 css 속성값이 바뀔 때 해당 변화를 일정 시간동안 일어나도록 해서 애니메이션을 만든다.
  • css 속성값은 수치가 있어야하고 animation이 적용가능 한지 확인해야한다.
  • 해당 요소에 스타일 속성이 바뀌는 방법은 제한이 없다.(class 추가,삭제,js 직접 지정...)

transition 속성들

  • transition-property: 변화를 감지할 속성, 애니메이션을 적용할 속성(default all)
  • transition-duration: 애니메이션이 일어나는 시간으로 s or ms 로 작성 (default 0)
  • transition-delay:변화를 감지한 후 애니메이션을 시작하는 시간
  • transition-timing-function: 애니메이션이 동작하는 속도
