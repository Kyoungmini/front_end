css 소개 및 사용법

css 로 할 수 있는 일

  1. document의 스타일 지정한다.
  2. document object(노드 개체)를 보이지 않게 할 수 있다.
  3. 노드 개체에 애니메이션을 지정할 수 있다.
  4. 외부 기기의 해상도에 따라 노드개체의 스타일을 변경할 수 있다.(크로스브라우징)
  5. canvas를 이용해 그림을 그릴 수 있다.

css 적용 법

css는 적용 우선순위를 갖고 우선 순위를 이용해 css 파일을 재사용하거나 자바스크립트로 css를 적용해(inline) 인터렉티브한 디자인을 만든다.

  1. inline : 요소에 직접 작성 (우선 순위가 가장 높다)
  2. internal : docuemnt에 작성된 style 태그안의 css로 exteral 보다 우선 순위가 높다.
    1. id 선택자(#)가 가장 우선 순위가 가장 높다.
    2. class(: or :: 의사선택자포함) 요소 선택자보다 우선 순위가 높다.
    3. tagName : *전체 선택자보다 우선 순위가 높다.
    4. * : 전체 선택자는 우선 순위가 가장 낮다.
  3. exteral : link로 internal 상단에 참조한 외부 파일로 참조한 순서대로 우선순위가 낮다.
