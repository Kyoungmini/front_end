노드의 컨텐츠가 노드의 범위를 넘을 때 보이게 할지 정하는 속성 Overflow

  • visible(default) : 컨텐츠가 요소의 박스를 넘어갈때 넘어간 내용을 표시한다.
  • hidden : 박스를 넘어간 내용을 표시하지 않는다.
  • scroll : 박스에 스크롤이 추가된다.
  • auto : 컨텐츠가 박스를 넘어가면 스크롤이 추가된다.

overflow CSS 단축 속성은 요소의 콘텐츠가 너무 커서 요소의 블록 서식 맥락에 맞출 수 없을 때의 처리법을 지정합니다. overflow-x (en-US), overflow-y (en-US)의 값을 설정합니다.

컨텐츠가 노드의 범위를 넘어가지 않는 이유

  • 브라우저의 인터페이스가 긴 dom을 보기 위해 스크롤을 내리는 방식으로 이루어진다.=>document의 길이에 제한이 없다.
  • document 노드개체의 길이는 auto로 content가 추가되면 계속 길이가 늘어난다.=>노드개체가 content를 포함하려 노력한다.
  • 보통 노드의 길이(height)를 고정하게 되면 content가 박스를 넘어가게 된다. (overflow,white-space)
