CSS Media Query 와 반응형 웹 디자인

반응형 웹 디자인이란?(RWD)

  • Responsive Web Design은 브라우저가 접속하는 기기의 화면 크기에 맞게 웹사이트의 화면의 ux ui가 바뀌는 것
  • 브라우저에 접속하는 다양한 기기의 발전으로 (모니터,스마트폰,테블릿..) 작거나 큰 화면에서도 문제가 없는 ui를 제공하기 위해 등장한 개념
  • RWD를 구축하면 url 하나로 여러 기기에서 접속하기 때문에 사용자 접속 관리가 용이하다.
  • 여러 기기에 대응되는 복수의 페이지를 생성할 필요가 없기 때문에 유지보수를 줄일 수 있다.
  • 한개의 페이지가 여러 기기에 대응 되야 하기 때문에 페이지 내의 css가 복잡해지고 고급 기술이 사용되어 개발 속도가 느린 편이다.(여러 웹사이트를 만드는 것보다 빠르지만 RWD를 하지 않으면 빠르게 한개의 사이트를 구축하고 서비스 할 수 있다.)

RWD를 위한 스타일

  • meta name="viewport" content="width=device-width, initial-scale=1.0"
    • 페이지의 크기와 배율을 제어
    • name="viewport" : 뷰포트를 제어하는 메타정보
    • initial-scale=1.0 : 브라우저에서 페이지를 처음 로드할 때 초기 확대 값
    • width=device-width : 브라우저를 로드하는 기기의 너비를 뷰포트의 너비로 설정
  • flex : 한줄의 레이아웃을 정의하는 display속성으로 flex로 선언된 컨테이너의 아이템들은 컨테이너 크기에 맞게 조절된다.
  • grid : 여러줄의 레이아웃을 table 구조로 정의하는 display 속성으로 flex 처럼 grid 내의 아이템들이 부모의 크기에 맞게 조절되고 아이템의 위치를 세부적으로 조절 가능해 layout을 구성하고 미디어쿼리로 구성된 layout을 화면기기에 맞게 조절하기 용이하다.
  • 단위 vw,vh,% : 뷰포트의 크기나 상대적 비율에 따라 노드의 크기를 변경할 수 있다.
  • max,min-width : 뷰포트의 크기를 범위로 지정할 수 있다.
  • @media : 뷰포트의 너비나 높이(장치의 너비와 높이) or 장치의 방향(가로,세로)에 따라 css를 다르게 적용할 수 있다.

미디어쿼리 문법

  • @media(조건){선택자{}}
  • 조건 width, height : viewport(스크롤포함) 길이
  • 조건 min-width, min-height : viewport(스크롤포함) 최소 길이
  • 조건 max-width, max-height : viewport(스크롤포함) 최대 길이
