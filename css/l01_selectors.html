  • class . 클래스 복수 선택자
  • tagName 태그이름 복수 선택자

Combinator selectors 선택자 조합

DOM 각 오브젝트가 포함관계의 노드를 갖는 tree 구조로 선택자 간의 관계 부모, 형제, 자식, 자손으로 설명 할 수 있다.

  • " " space:
  • ">" :
  • "+" :
  • "~" :

Pseudo-classes 의사 선택자

노드 개체가 상태가 변하는 것에 따라 찾을 수 있는 선택자로 클래스와 우선 순위가 같다. selector:pseudo-class

  • :checked
  • :hover
  • :focus
  • :link
  • :visited
  • :active
  • :target
  • :required
  • :not(selector)
  • :nth-child(n)

Pseudo-Elements 의사 요소 선택자

노드 개체 내부의

  • ::after
  • ::before
  • ::first-letter
  • ::first-line
  • ::selection

Attribute Selectors 속성 선택자

특정 속성이나 속성의 값으로 노드 개체를 찾는 선택자

  • [attribute]
  • [attribute="value"]
  • [attribute~="value"
  • [attribute|="value"]
  • [attribute^="value"]
  • [attribute$="value"]
  • [attribute*="value"]
