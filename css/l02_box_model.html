Box
  • width (max,min): content 가로 길이(px,em,%,vw .....)
  • height(max,min): content 세로 길이
  • padding:content와 border 사이의 공간 (top,right,bottom,left)
  • border: box model의 테두리로(실제크기) 두께와 스타일과 색을 지정할 수 있다. (top,right,bottom,left)
    • -width: 테두리의 두께
    • -style: 테두리의 스타일(solid,dotted,double,groove..,none,hidden)
    • -color: 테두리의 색
    • -radius:테두리를 둥글게(content,padding,outline모두 같이 둥글게 된다.),%는 box를 둥글게 만든다.
  • outline: 테두리와 여백 사이의 윤관선으로 box model에 포함되지 않아 크기에 적용되지 않고 마이너스를 적용하면 content까지 줄어들 수 있다.
    • -width: 윤곽선의 두께
    • -style: 윤곽선의 스타일
    • -color: 윤곽선의 색
    • -offset:윤곽선과 border 사이의 여백
  • margin: box model의 여백으로 block은 left,right에 auto로 box를 정렬할 수 있다.
  • box-shadow:
  • box-sizing: width와 height를 content에서 border까지로 변경하는 속성
    • content-box(default):width와 height가 content 영역의 크기를 지정
    • border-box: width와 height가 border 영역의 크기를 지정
