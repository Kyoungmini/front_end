양식을 제출하는 form 요소

파라미터를 서버와 통신할때 포함시키기 위해 form 양식 요소를 제공하고 있다.

  • action : 서버의 url
  • method : 서버와 통신하는 방법 default get
    • get: 특정페이지를 요청할때 페이지의 정보를 다르게 하기 위해 사용 ?param=value¶m=value&.... 예)naver.com/앨범?search="산"&page=4;
    • post: 양식을 처리하는 페이지를 요청할 때 사용하고 처리 상태에 따라 다른 페이지를 서버에서 제공한다. 파리미터는 html 문서 해더 정보에 포함된다. 예)회원가입,로그인,설문조사제출,회원정보수정,삭제....
    • target : 양식 제출 시 받은 응답을 표시할 창의 위치
      • _self(default) :동일한 창에 표시
      • _blank : 새 창이나 프레임에 표시
      • _parent : 상위 프레임에 표시
      • _top : 창 전체에 표시
  • name : document에서 유일한 이름으로 js에서 document.forms에서 참조한다.
