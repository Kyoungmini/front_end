수업준비

  • ul은 목록 태그입니다.
  • 비쥬얼스튜디오 코드(VSC)는 편집 프로그램이다.
  • VSC에 Extensions는 확장 프로그램을 설치할 수 있다.
  • Extensions의 live server는 작성한 문서를 서버프로그램에 올려서 저장할 때 마다 새로고침 해주는 편리한 도구다.

HTTP(HyperText Transfer Protocol)와 HTML(HyperText Markup Language)

  1. html은 http 통신을 할 때 요청(클라이언트,브라우저)하는 페이지를 서버가 응답하는 문서이다.
  2. http 통신은 서버와 클라이언트가 통신을 하는 규약으로 요청과 응답의 형식을 제공한다.

HTML 기본 작성법

  1. html은 마크업 언어로 태그를 만들고 내용을 작성한다.
  2. html 태그는 약속된 이름이 존재하고 열린태그는 반드시 닫아야한다. < >열고 < / > 닫는다
  3. html 문서는 트리구조로 최소위 태그로 html이 존재해야 한다.
  4. 태그는 소문자로 작성하는 것을 권장하고 약속된 이름만 사용한다.
  5. HTML5는 가장 최신 html 문서 규칙으로 문서 내부에 오류를 자동으로 수정한다.
html 외부에 작성된 내용은 오류로 간주되어 자동으로 내부로 편입된다.
