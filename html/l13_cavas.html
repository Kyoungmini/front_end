html canvas

cavas는 웹 페이지에 그래픽을 그려주는 컨테이너다. canvas는 자바스크립트로 그래픽을 그린다.

strokeRect(x1,y1,x2,y2) 사각형 그리기

  • fillStyle : 색이 있는 사각형 색 지정
  • fillRect : 색이 있는 사각형 그리기
  • strokeRect : 사각형 테두리 그리기
  • clearRect : 사각형 모양의 지우개
웹브라우저가 canvas를 지원하지 않습니다!(라는 문자만 출력)

lineTo 선그리기

  • moveTo : 선이 시작될 좌표
  • lineTo : 선이 끝나는 좌표(연속으로 지정할 수 있음)
  • stroke : 선 그리기 시작

Colors, Styles, Shadows 속성 주기

  • fillStyle : 색, 공간기울기색(gradient),패턴을 채우는 설정
  • strokeStyle : 색, 공간기울기색(gradient),패턴을 그리는 설정(lineWidth로 두께 설정)
  • shadowColor : 그림자 색
  • shadowBlur : 그림자 흐려짐 정도
  • shadowOffsetX : 그림자 수평 이동(양수:오른쪽,음수:왼쪽)
  • shadowOffsetY : 그림자 수직 이동(양수:아래쪽,음수:윗쪽)

Gradient 번지는 색(기울기색)

createLinearGradient(x0,y0,x1,y1)

  • x0 :그래디언트 시작점 x 좌표
  • y0 :그래디언트 시작점 y 좌표
  • x1 :그래디언트 끝점 x 좌표
  • y1 :그래디언트 끝점 y 좌표
