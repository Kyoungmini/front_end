jqeury 사용방법

  • $("css선택자") : 통합된 document 선택자로 모두 복수로 선택 한다.
  • $(function(){}),$(document).ready() == window.onload : document가 모두 load된 시점
  • .css() : .style 해당 노드 개체에 직접 지정한 style만 찾을 수 있는데 .css는 해당 노드 개체의 style의 상태를 알 수 있다.
  • jquery에서 window 대신에 #document 개체를 사용
  • chaining : $().css().click().animate().... 선택자에 박복적으로 함수를 지정할 수 있다.
