jqeury를 이해하고 사용해보자

  • cdn jqeury link 사용할때 압축되지 않은 것을 사용
  • $ : jqeury 최상위 객체로 $를 사용하는 framework에서는 이름을 바꿔야한다.
  • $() : ==document.querySelectorAll() 처럼 dom을 검색하는 도구 return jqueryCollection
  • jQeury.fn.init(jqueryCollection)에는 자바스크립트 document에서 제공하는 함수와 이름이 비슷한 함수를 포함한다. 예)onclick=>click,styel=>css
  • jquery 함수는 반복을 자동으로 포함하고 있다.
  • jquery 에는 javascript에서 제공하지 않는 함수도 존재한다. (hover(function,function))
  • 이벤트함수는 on을 제거하면 모두 존재한다.
  • on("이벤트이름",function)으로 작성가능
  • 자바스크립트 dom에 jqeury 함수를 사용할 수 없다 예) this.css("color","red")(x)=> $(this).css("color","red")(o)
